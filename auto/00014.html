OSS UI Tools for HTML Rendering

If you want an experience like Claude's "Artifacts" or a built-in sandbox for HTML/JS previewing, these are the top open-source contenders.

Lobe Chat

Extremely polished UI. It supports a plugin system and features a Code Preview mode that renders HTML/CSS/JS directly in the chat window.

  • Modern UX/UI
  • Artifact-like previews
  • PWA Support
View GitHub

OpenCanvas

Developed by LangChain. It is explicitly built to mimic the Claude Artifacts experience, focusing on side-by-side editing and rendering of code/documents.

  • Specialized in Code/HTML
  • Real-time preview panel
  • History versioning
View GitHub

Bolt.new (OSS)

This is the ultimate tool for full-stack HTML/React rendering. It runs a full WebContainer in your browser to build and preview apps live.

  • Full environment sandbox
  • Multi-file support
  • Instant deployment links
View GitHub

LibreChat

A powerful alternative to ChatGPT. While general-purpose, it has a robust Code Interpreter and plugin system for rendering visualizations and HTML.

  • Enterprise grade Auth
  • Plugin Architecture
  • Multi-model support
View GitHub

Chatbox

A desktop/web client that offers very clean code block management and simple web preview capabilities for quick HTML snippets.

  • Native Desktop Apps
  • Local model support
  • Simple & Fast
View GitHub

Open WebUI (Customized)

Don't switch yet! Open WebUI supports "Functions" and "Tools" where you can install community "Artifacts" filters that enable HTML rendering.

  • Community Tooling
  • Most features of any OSS
  • Self-host easy via Docker
Explore Functions

Pro-tip: If you want 1:1 Claude Artifacts behavior, OpenCanvas is your best bet. If you want a full development suite, look at Bolt.new.
